# NIFTY-50 Investment Intelligence Platform

An AI-powered investment decision-support system built on historical NSE NIFTY-50 market data (January 2000 – April 2021). The platform combines machine learning, financial analytics, portfolio optimisation, and risk assessment into an interactive Streamlit web application.

## Environment Setup

### Prerequisites

- Python 3.10, 3.11, or 3.12  
  *(Python 3.13 is not recommended — TensorFlow is incompatible)*
- pip

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/investment-intelligence-nifty50.git
cd investment-intelligence-nifty50
```

### 2. Create a virtual environment (recommended)

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS / Linux
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Download the dataset

Download the NIFTY-50 dataset from Kaggle:

**Primary dataset:** https://www.kaggle.com/datasets/rohanrao/nifty50-stock-market-data/data

Extract all CSV files into the `data/` folder. The folder must contain:
- 50 individual stock CSVs (e.g. `RELIANCE.csv`, `TCS.csv`)
- `stock_metadata.csv`
- `NIFTY50_all.csv`

## Running the Application

```bash
streamlit run app/streamlit_app.py
```

Opens at `http://localhost:8501` with 6 pages:

| Page | Description |
|------|-------------|
| Overview | Platform summary and sector breakdown |
| Stock Analyser | Candlestick chart, RSI, MACD, 52-week stats |
| AI Predictor | Ensemble predictions, buy/sell signals for any stock |
| Portfolio Builder | Investor-profile portfolios, efficient frontier, share allocation |
| Risk Assessment | VaR, CVaR, stress testing, rolling risk, risk contribution |
| Market Overview | 1-year return heatmap, correlation matrix, risk scorecard |

> **First run note:** If saved models are not present, the AI Predictor trains on demand (8–12 minutes for all 50 stocks). Run Notebook 03 first to pre-train all models.

## Reproducing Results

Run notebooks in order. Each saves outputs used by subsequent notebooks and the app.

```bash
jupyter notebook
```

| Step | Notebook | Saves |
|------|----------|-------|
| 1 | `01_EDA.ipynb` | Charts in `report/` |
| 2 | `02_feature_engineering.ipynb` | `data/master_features.parquet`, `models/feature_cols.pkl` |
| 3 | `03_stock_predictor.ipynb` | `models/<SYMBOL>_*.pkl`, `models/latest_signals.csv` |
| 4 | `04_portfolio_construction.ipynb` | `models/portfolios.json` |
| 5 | `05_risk_assessment.ipynb` | `models/risk_scorecard.csv` |

Then launch the app:

```bash
streamlit run app/streamlit_app.py
```

## Key Results

### Stock Predictor — Ensemble (XGBoost + Random Forest)

- **5-day directional accuracy:** 52–55% across NIFTY-50 universe
- **Return forecast MAE:** ~1.8–2.4% over 5-day horizon
- Evaluated using 5-fold walk-forward time-series cross-validation (no lookahead bias)

### Portfolio Construction (2015–2021 backtest)

| Profile | Strategy | Annual Return | Volatility | Sharpe |
|---------|----------|:---:|:---:|:---:|
| Conservative | Min Volatility | 5.6% | 14.3% | 0.04 |
| Balanced | Max Sharpe Ratio | 22.5% | 18.5% | 0.95 |
| Aggressive | Max Return | 28.3% | 22.7% | 1.03 |
| Risk Parity | Inverse Volatility | 10.9% | 17.8% | 0.33 |
| Equal Weight | Benchmark | 10.7% | 18.5% | 0.31 |

### Risk Assessment

- Metrics: Sharpe, Sortino, Calmar, Max Drawdown, VaR 95/99%, CVaR, Beta, Alpha
- Stress tests: 2011 Euro Debt Crisis, 2015 China Slowdown, 2020 COVID Crash, 2020 Recovery
- Rolling 126-day risk metrics for dynamic monitoring

## Dataset

| Property | Detail |
|----------|--------|
| Source | NSE NIFTY-50 (Kaggle) |
| Period | January 2000 – April 2021 |
| Stocks | 50 companies |
| Sectors | 13 sectors |
