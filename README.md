# NIFTY-50 Investment Intelligence Platform

An AI-powered investment decision-support system built on historical NSE NIFTY-50 market data (January 2000 – April 2021). The platform combines machine learning, financial analytics, portfolio optimisation, risk assessment, explainability, and anomaly detection into an interactive 8-page Streamlit web application.

## Environment Setup

### Prerequisites
- Python 3.10, 3.11, or 3.12  *(Python 3.13 not recommended — TensorFlow incompatible)*
- pip

### 1. Clone the repository
```bash
git clone https://github.com/agcode-maker/investment-intelligence-nifty50.git
cd investment-intelligence-nifty50
```

### 2. Create a virtual environment (recommended)
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS / Linux
source venv/bin/activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Download the dataset
Download from Kaggle: https://www.kaggle.com/datasets/rohanrao/nifty50-stock-market-data/data

Extract all CSV files into the `data/` folder. Required files:
- 50 individual stock CSVs (e.g. `RELIANCE.csv`, `TCS.csv`)
- `stock_metadata.csv`
- `NIFTY50_all.csv`

## Running the Application

```bash
streamlit run app/streamlit_app.py
```

Opens at `http://localhost:8501` — 8 interactive pages:

| Page | Description |
|------|-------------|
| 🏠 Overview | Platform summary, sector composition, capability overview |
| 🔍 Stock Analyser | Candlestick + RSI + Volume + MACD, toggleable indicators, full risk stats |
| 🤖 AI Predictor | Run ensemble model per stock, probability gauge, buy/sell signal chart, all-stocks signal table |
| 💼 Portfolio Builder | 4 investor profiles, efficient frontier, sector exposure, ₹ discrete allocation |
| ⚠️ Risk Assessment | VaR/CVaR, rolling risk, portfolio comparison, risk contribution, stress test |
| 📊 Market Overview | 1-year return heatmap, correlation matrix, full risk scorecard |
| 🧠 Explainable AI | SHAP global importance, local prediction waterfall, SHAP over time |
| 🚨 Anomaly Detection | Volatility spikes, extreme moves, volume anomalies, Isolation Forest, universe scan |

> **First run:** If saved models are not present, the AI Predictor trains on demand (~8–12 min for all stocks). Run Notebook 03 first to pre-train all models.

## Key Results

### Stock Predictor — Ensemble (XGBoost + Random Forest)

| Metric | Value |
|--------|-------|
| 5-day Directional Accuracy | 52–55% across 49 stocks |
| Return Forecast MAE | ~1.8–2.4% over 5-day horizon |
| Validation Method | 5-fold walk-forward time-series CV (no lookahead bias) |

### Portfolio Construction (2015–2021 backtest)

| Profile | Strategy | Annual Return | Volatility | Sharpe | Holdings |
|---------|----------|:---:|:---:|:---:|:---:|
| Conservative | Min Volatility | 5.6% | 14.3% | 0.04 | 21 |
| Balanced | Max Sharpe | 22.5% | 18.5% | 0.95 | 9 |
| Aggressive | Max Return | 28.3% | 22.7% | 1.03 | 4 |
| Risk Parity | Inverse Vol | 10.9% | 17.8% | 0.33 | 48 |
| Equal Weight | Benchmark | 10.7% | 18.5% | 0.31 | 48 |

### Risk Assessment
- Metrics: Sharpe, Sortino, Calmar, Max Drawdown, VaR 95/99%, CVaR, Beta, Alpha
- Stress tests: 2015 China Slowdown, 2020 COVID Crash, 2020 Recovery
- Rolling 126-day risk metrics for dynamic monitoring

### Anomaly Detection
- Methods: Volatility z-score, Extreme move threshold, Volume z-score, Isolation Forest
- Universe anomaly rate: 6.9–8.9% of trading days flagged per stock
- Crisis validation: anomaly clustering confirmed in 2008 GFC and 2020 COVID

### Explainable AI (SHAP)
- Global feature importance across 300+ recent trading days
- Local waterfall explanation for every individual prediction
- Top features: Return_1d, Rolling_Std_5, HV_21, RSI, Price_vs_SMA20
